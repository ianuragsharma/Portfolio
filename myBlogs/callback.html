Callback Functions in JavaScript

What is a callback function

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.

Lets break this down and find out what is it with a simple example below: 

        function callback(name) {
            console.log("Hello",name)
          }
          
          function outer(callback) {
            const name = prompt('Please Type your name');
            callback(name);
          }
          
          outer(callback);

Still not clear with it? lets take another example

        const callback = function() {  
            console.log("This message is shown after 3 seconds");
        }
         
        setTimeout(callback, 3000);

Here setTimeout function is calling the callback function after 3000 miliseconds.This is a simple example of callback.So basically function that is passed to another function as a parameter is a callback function
