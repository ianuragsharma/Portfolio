Callback Functions in JavaScript

What is a callback function

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.

Lets break this down and find out what is it with a simple example below: 

  function callback(name) {
    console.log("Hello",name)
  }      
  function outer(callback) {
    const name = prompt('Type your name');
    callback(name);
    }
    outer(callback);

Still not clear with it? lets take another example

  const callback = function() {
    console.log("This message is shown after 3 seconds");
    }
  setTimeout(callback, 3000);

Here setTimeout function is calling the callback function after 3000 miliseconds.This is a simple example of callback.So basically function that is passed to another function as a parameter is a callback function

But Why use callback?

Simple! Callbacks helps in running the JavaScript asynchronously.Callbacks make sure that a function is not going to run before a task is completed but will run right after the task has completed.As we know javascript is a single threaded language synchronous language Callback function opens up a whole new world to explore

Lets see an example.

  < button id="btn">Click here< /button>
      
  document.queryselector("#btn")
    .addEventListener("click", function() {    
      console.log("Button Clicked");
});

In the first code snipped we create a button with the id btn

In the second code snipped we select that button and with the help of that id and add an event listner to it.The addEventListener essentially takes two parameters first "click" event and the second parameter is nothing but a callback function.
